Analysis on Netflix Movies & TV Shows

In this EDA, I will explore the netflix-shows dataset through visualizations and graphs using matplotlib and seaborn. Netflix Data Analysis with Python The dataset I use for the Netflix data analytics task consists of TV shows and movies streamed on Netflix as of 2021. The dataset is provided by Kaggle.com .

Package Install and Import

First, we will install and import necessary packages.: pandas, numpy, matplotlib, and seaborn.

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
%matplotlib inline
import seaborn as sns

Handling Null Values :

df.isna()

df.isna().sum()

As we can see that in the title,description,age_certification,seasons,imdb_id,imdb_score,_imdb_votes,tmdb_popularity and tmdb_score has null values so, we will show this with the help of heatmap.

plt.figure(figsize=[10,6])
sns.heatmap(df.isna())

we will be filling the missing values of title,description,age_certification,imdb_id, with unavailable.

df.fillna({'title':'Unavailable','description':'Unavailable','age_certification':'Unavailable','imdb_id':'Unavailable'},inplace=True)
df.isna().sum()

Now we are done with missing values, but the imdb_score,imdb_votes,tmdb_popularity and tmdb_score are still not quite right… So we will be filling the missing values of imdb_score,imdb_votes,tmdb_popularity and tmdb_score with mean of that column by mean() function.

df.fillna({'imdb_score':'df.imdb_score.mean()','imdb_votes':'df.imdb_votes.mean()','tmdb_popularity':'df.tmdb_popularity.mean()','tmdb_score':'df.tmdb_score.mean()'},inplace=True)
df.isna().sum()

df['seasons'].fillna(value=1,inplace=True)
df.isna().sum()

Netflix Film Ratings :

Now, we will explore the ratings which are based on the film rating system. The ordering of the ratings will be based on the age of the respective audience from youngest to oldest. We will not include the ratings 'NR' and 'UR' in the visuals since they stand for unrated and non-rated content.

df.age_certification.value_counts()

order = ['G', 'TV-Y', 'TV-G', 'PG', 'TV-Y7','TV-PG', 'PG-13', 'TV-14', 'R', 'NC-17', 'TV-MA']
plt.figure(figsize=(15,7))
sns.countplot(df.age_certification,hue=df.type,order=order,saturation=0.8);
plt.xlabel("Rating")
plt.title("Ratings for Movies & Shows")

fig, ax = plt.subplots(1,2, figsize=(19, 5))
g=sns.countplot(df.age_certification, order=order, ax=ax[0]);
g.set_title("Ratings for Movies")
g.set_xlabel("Rating")
f=sns.countplot(df.age_certification, order=order, ax=ax[1]);
f.set(yticks=np.arange(0,1600,200))
f.set_title("Ratings for TV Shows")
f.set_xlabel("Rating")
plt.show()

The shows and movies with a rating of TV-MA (Mature Audience Only) are in the clear majority.
